#DataTypes
Основные типы данных в C++:
- int: целые числа
- float: числа с плавающей точкой одинарной точности
- double: числа с плавающей точкой двойной точности
- char: символы
- bool: логические значения (true/false)
- unsigned, signed: модификаторы для типов
- long, short: модификаторы длины

#PointersAndReferences
Ссылки и указатели в C++:
- Указатели (pointers): хранят адрес памяти переменной. С помощью указателя можно изменять значение по адресу.
Пример:
  int x = 10;
  int* ptr = &x; // указатель на x
  *ptr = 20; // изменяем значение x через указатель

- Ссылки (references): создают альтернативное имя для существующей переменной.
Пример:
  int y = 10;
  int& ref = y; // ссылка на y
  ref = 20; // изменяем значение y через ссылку
